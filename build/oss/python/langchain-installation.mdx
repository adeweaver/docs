To install the core LangChain package

```bash
pip install -U langchain
```




LangChain also provides integrations to 100s of LLMs and thousands of other integrations.
In order to use a specific provider/integration, you'll need to install the corresponding package.

```bash
# installing the OpenAI integration
pip install -U langchain-openai
```




See the [integrations page](/oss/python/js-integrations) for a full list of available integrations.
