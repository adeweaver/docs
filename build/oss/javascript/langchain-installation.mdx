To install the core LangChain package



<CodeGroup>
```bash npm
npm install langchain
```

```bash pnpm
pnpm add langchain
```

```bash yarn
yarn add langchain
```

```bash bun
bun add langchain
```
</CodeGroup>


LangChain also provides integrations to 100s of LLMs and thousands of other integrations.
In order to use a specific provider/integration, you'll need to install the corresponding package.



<CodeGroup>
```bash npm
# installing the OpenAI integration
npm install @langchain/openai
# installing the Anthropic integration
npm install @langchain/anthropic
```

```bash pnpm
# installing the OpenAI integration
pnpm add @langchain/openai
# installing the Anthropic integration
pnpm add @langchain/anthropic
```

```bash yarn
# installing the OpenAI integration
yarn add @langchain/openai
# installing the Anthropic integration
yarn add @langchain/anthropic
```

```bash bun
# installing the OpenAI integration
bun add @langchain/openai
# installing the Anthropic integration
bun add @langchain/anthropic
```
</CodeGroup>


See the [integrations page](/oss/javascript/js-integrations) for a full list of available integrations.
