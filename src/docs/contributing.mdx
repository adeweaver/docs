Welcome! Thank you for your interest in contributing.

LangChain has many widely-used open-source projects. Being in a rapidly developing field, we are extremely open to contributions. Whether you're fixing bugs, adding features, improving documentation, or sharing feedback, you can help make LangChain better for everyone.

## Ways to Contribute

### Report Bugs

Found a bug? Please help us fix it:

1. Check if the issue already exists in our [GitHub Issues](https://github.com/langchain-ai) for the respective repo.
2. If not, create a new issue with following the issue template provided for the project. Be sure to include any relevant labels.

### Suggest Features

Have an idea for a new feature or enhancement?

1. Search our [forum](https://forum.langchain.com/) for existing feature requests
2. If it doesn't exist, start a new dicussion under the relevant category so that project maintainers and the community can provide feedback.
3. Be sure to describe the use case and why it would be valuable.
