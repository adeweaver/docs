**Welcome! Thank you for your interest in contributing.**

Being in a rapidly developing field, we are extremely open to contributions to our OSS projects. Whether you're fixing bugs, adding features, improving documentation, or sharing feedback, you can help make LangChain and LangGraph better for everyone.

## Ways to Contribute

<AccordionGroup>
    <Accordion title="Report bugs" icon="bug">
        Found a bug? Please help us fix it by following these steps:

        <Steps>
            <Step title="Search">
                Check if the issue already exists in our GitHub Issues for the respective repo:
                {/* TODO fix the icon backgrounds and fill colors to respect theme */}
                :::python
                <Card title="LangChain" icon="/images/brand/langchain.svg" href="https://github.com/langchain-ai/langchain" />
                <Card title="LangGraph" icon="/images/brand/langgraph.svg" href="https://github.com/langchain-ai/langgraph" />
                :::
                :::js
                <Card title="LangChain" icon="/images/brand/langchain.svg" href="https://github.com/langchain-ai/langchainjs" />
                <Card title="LangGraph" icon="/images/brand/langgraph.svg" href="https://github.com/langchain-ai/langgraphjs" />
                :::
            </Step>
            <Step title="Create Issue">
                If not, create a new issue with following the issue template provided for the project. Be sure to include any relevant labels.
            </Step>
            <Step title="Wait">
                A project maintainer will triage the issue and may ask for additional information. Please be patient as we manage a high volume of issues.
            </Step>
        </Steps>
    </Accordion>

    <Accordion title="Suggest features" icon="wand-magic-sparkles">
        Have an idea for a new feature or enhancement?

        <Steps>
            <Step title="Search">
                Search our [forum](https://forum.langchain.com/) for existing feature requests
            </Step>
            <Step title="Discuss">
                :::python
                If it doesn't exist, start a new dicussion under the [relevant category](https://forum.langchain.com/c/help/langchain/14) so that project maintainers and the community can provide feedback.
                :::
                :::js
                If it doesn't exist, start a new dicussion under the [relevant category](https://forum.langchain.com/c/help/langgraph/13) so that project maintainers and the community can provide feedback.
                :::
            </Step>
            <Step title="Describe">
                Be sure to describe the use case and why it would be valuable to others.
            </Step>
        </Steps>
    </Accordion>

    <Accordion title="Improve documentation" icon="book">
        Documentation improvements are always welcome! We strive to keep our docs clear and comprehensive, and your perspective can make a big difference.

        <Card title="Documentation guide" href="/oss/contributing/documentation" arrow />
    </Accordion>

    <Accordion title="Contribute code" icon="code">
        With a wide user base, it can be hard for our small team to keep up with all the feature requests and bug fixes. If you have the skills and time, we would love your help!

        <Card title="How to make your first PR" href="/oss/contributing/code" arrow />
    </Accordion>
</AccordionGroup>

Thank you for helping make LangChain better!
